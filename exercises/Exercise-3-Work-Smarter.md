# Exercise 3: Work Smarter — Plan a Joint Readiness Review
**Module 3 | Estimated Time: 2 hours**

---

## Scenario
The USINDOPACOM Commander has directed a **Joint Readiness Review (JRR)** to assess theater-wide force readiness and preparedness for potential contingencies. You are a staff action officer tasked with planning the review, coordinating across directorates, and producing the supporting materials. You will use Copilot Chat across multiple Microsoft 365 apps to complete this work.

---

## Task 3.1 — Research and Brainstorm (20 min)

### Part A: Research Readiness Frameworks

In **Copilot Chat** (browser or Teams), enter:

```
I am a staff action officer at USINDOPACOM planning a Joint Readiness
Review for the Commander. Summarize the key components of a military
readiness review, including personnel readiness, equipment readiness,
training readiness, and logistics readiness. For each component,
describe what is typically assessed and what metrics are used.
Present as a structured brief with headers.
```

### Part B: Brainstorm Agenda Topics

```
Based on the readiness components above, brainstorm 10 agenda topics
for a two-day Joint Readiness Review. The audience includes the
INDOPACOM Commander, component commanders, and directorate leads
(J1 through J8). Include both briefing topics and working group
sessions. Present as a numbered list with a one-sentence description
for each topic.
```

### Part C: Narrow to Final Agenda

```
From the 10 topics above, select the 6 most critical for a Pacific
theater-focused readiness review. Organize them into a two-day agenda
with morning and afternoon blocks. Include 15-minute breaks and a
working lunch. Present as a table with columns: Day, Time Block,
Topic, Lead Directorate, and Format (Brief/Discussion/Working Group).
```

## Task 3.2 — Create the Read-Ahead Document in Word (25 min)

Open **Microsoft Word** and activate Copilot Chat from the ribbon (Copilot icon, upper right).

### Part A: Draft the Read-Ahead

In the Copilot Chat side pane within Word, enter:

```
Draft a Joint Readiness Review read-ahead document for USINDOPACOM.
The document should include:
1. Purpose — explain why the review is being conducted
2. Background — 2 paragraphs on the current operating environment
   in the Indo-Pacific
3. Scope — what the review will and will not cover
4. Agenda Overview — summarize the two-day agenda
5. Key Questions for Discussion — list 5 questions the Commander
   wants answered
6. Administrative Details — date, location, classification, and
   POC information

Use formal military writing style. Use fictitious but realistic
details. Date the review for 22-23 October 2026 at Camp H.M. Smith,
Hawaii.
```

### Part B: Refine the Document

Use follow-up prompts to improve specific sections:

```
Rewrite the Background section to emphasize the importance of allied
interoperability and the increasing pace of operations in the
Second Island Chain.
```

```
Add a section titled "Readiness Metrics Dashboard" that describes
a fictitious readiness scorecard with categories rated Green, Amber,
or Red. Include Personnel, Equipment, Training, Logistics, and
Cyber Readiness.
```

### Part C: Copy to Your Document

Select and copy the best version of each section into your Word document. Save the file as **JRR-Read-Ahead-2026.docx**.

## Task 3.3 — Build the Budget Tracker in Excel (20 min)

Open the provided file **assets/JRR-Budget-Data.csv** in **Microsoft Excel**. Then open the Copilot Chat side pane.

### Part A: Analyze the Data

```
Review the data in this spreadsheet. Summarize the total budget
allocation and spending by directorate. Which directorate has the
highest spend and which has the most remaining budget?
```

### Part B: Create Visualizations

```
Create a bar chart comparing budget allocation versus actual spend
for each directorate.
```

### Part C: Identify Risks

```
Based on the spending data, identify any directorates that are at
risk of exceeding their budget. What recommendations would you
make to the J8 (Resources) lead?
```

### Part D: Add a Projection

```
Add a column that projects end-of-year spending based on the current
burn rate. Flag any directorate projected to exceed 100% of their
allocation.
```

## Task 3.4 — Draft the Invitation in Outlook (15 min)

Open **Microsoft Outlook** and activate Copilot Chat from the ribbon.

### Part A: Draft the Meeting Invitation

```
Draft a formal email from the USINDOPACOM Chief of Staff to all
component commanders and J-directorate leads inviting them to
the Joint Readiness Review on 22-23 October 2026 at Camp H.M. Smith.
The email should convey the importance of attendance, reference the
read-ahead document, request that each directorate prepare a
10-minute readiness status brief, and include an RSVP deadline
of 8 October 2026. Use formal military correspondence style.
```

### Part B: Refine

```
Make the tone slightly less formal — this is going to senior leaders
who the Chief of Staff works with daily. Keep it professional but
collegial. Add a line about a hosted dinner on the evening of
22 October.
```

Send the email to yourself as a test.

## Task 3.5 — Create the Briefing Deck in PowerPoint (20 min)

Open **Microsoft PowerPoint** and activate Copilot Chat from the ribbon.

### Part A: Generate Slides

```
Create a 6-slide briefing for the Joint Readiness Review at
USINDOPACOM, 22-23 October 2026. Include:
Slide 1: Title slide with event name, dates, location
Slide 2: Purpose and scope
Slide 3: Current readiness posture overview
Slide 4: Key readiness metrics (Personnel, Equipment, Training,
         Logistics, Cyber) with Green/Amber/Red status
Slide 5: Priority actions and recommendations
Slide 6: Next steps and timeline
Use a professional, clean design.
```

### Part B: Prepare for Questions

```
What are 5 challenging questions that component commanders might
ask during this readiness review? For each question, suggest a
concise talking point response.
```

Copy the Q&A to your notes or add as a hidden appendix slide.

## Task 3.6 — Collaborate Using Copilot Pages (15 min)

Return to **Copilot Chat** in the browser.

### Part A: Create a Planning Page

Enter this prompt:

```
Create a project tracker for the Joint Readiness Review with the
following columns: Task, Owner, Directorate, Due Date, Status.
Populate it with 10 realistic planning tasks such as "Finalize
agenda," "Distribute read-ahead," "Confirm AV/IT support,"
"Book conference rooms," etc. Set due dates working backward
from the 22 October 2026 event date.
```

### Part B: Convert to a Page

1. On Copilot's response, select **"Edit in Pages"**.
2. The response opens as an editable Copilot Page.
3. Edit the page:
   - Add your name as the owner for 2-3 tasks.
   - Change one task's status to "In Progress."
   - Add a new row: "Coordinate with Allied Liaison Officers."

### Part C: Share the Page

1. Select **Share** on the Copilot Page.
2. Share with a classmate via Teams or email.
3. Have them edit one task to demonstrate real-time collaboration.
