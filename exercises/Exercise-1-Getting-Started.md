# Exercise 1: Getting Started with Copilot Chat
**Module 1 | Estimated Time: 30 minutes**

---

## Scenario
You have just arrived at USINDOPACOM headquarters and need to get up to speed quickly using Microsoft 365 Copilot Chat as a daily productivity tool. In this exercise you will access Copilot Chat, explore the interface, and verify Enterprise Data Protection is active.

>Note: TRAINING SCENARIO ONLY. All names, dates, and data are fictitious. In real-world planning, you must ensure any information you input is unclassified and approved for use in this tool. When in doubt, ask your security officer.

---

## Task 1.1 — Access Copilot Chat (5 min)

> Note: If you cannot access Copilot, your unit's ISSM or CIO shop controls the licenses. Do not use a personal or commercial version for any work."

1. Open your web browser and navigate to **https://office365.us/chat**.
2. Sign in with your **work email account**.
3. Confirm you see the Copilot Chat home screen with the prompt input box.

**Try it:** Also open Copilot Chat from a second entry point:
- In **Microsoft Teams**, select the Copilot icon from the left-side menu.
- In the **Microsoft 365 Copilot mobile app** on your phone (if available).

> **Note (GCC High):** The Copilot icon in the Edge browser sidebar is not available in GCC High environments. Use the browser at https://www.office365.us/chat or Teams instead.

> **Question for discussion:** Which access point do you think you would use most frequently in your daily role? Why?

## Task 1.2 — Explore the Interface (10 min)

Familiarize yourself with the Copilot Chat interface by locating the following elements:

| Element | Where to Find It |
|---------|-----------------|
| Prompt input box | Bottom center of screen |
| Conversation history | Left sidebar |
| Add content button | Next to the prompt input box |
| Copilot Pages | Left sidebar or top navigation |
| Prompt suggestions | Displayed on home screen |

**Try it:** Select one of the suggested prompts displayed on the home screen and observe how Copilot responds.

---

## Task 1.3 — Your First INDOPACOM Prompt (10 min)

> **Note (GCC High):** Web grounding (Copilot's ability to search the internet for answers) is not enabled by default in GCC High. If your prompts return limited results, your admin may need to enable the "Allow web search in Copilot" policy. If web search is not available, Copilot will still respond using its built-in knowledge, but results may be less current.

Enter the following prompt into Copilot Chat:

```
What are three things I should consider when drafting talking points for a general officer about the humanitarian situation in U.S. Indo-Pacific Command. Keep the response under 200 words.
```

After receiving the response:

1. **Check the sources** listed beneath the answer. Click on at least one source link to verify the information.
2. **Ask a follow-up question:**

```
Now list the key allied nations that INDOPACOM partners with for
theater security cooperation. Present as a bulleted list.
```

3. **Refine the output:**

```
Rewrite that list as a two-column table with the country name in the
first column and the primary cooperation area in the second column.
```

> **Key takeaway:** Notice how you can iterate on Copilot's responses conversationally, just as you would refine a request to a staff member.

---

## Task 1.4 — Verify Enterprise Data Protection (5 min)

1. In the Copilot Chat interface, look for the **shield icon** or **"Protected"** indicator, which confirms Enterprise Data Protection (EDP) is active.
2. Enter this prompt:

```
What security protections are in place for my data when I use
Microsoft 365 Copilot Chat in my organization?
```

3. Review Copilot's response about EDP protections.

> **Important:** EDP ensures that your prompts and responses are not used to train AI models, are not visible to other users, and are processed within Microsoft's enterprise security boundary.
