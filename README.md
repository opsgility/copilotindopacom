| | |
|---|---|

| **Classification** | UNCLASSIFIED |
| **Environment** | GCC High |

## Course Overview

This course provides hands-on, scenario-based training for USINDOPACOM personnel on Microsoft 365 Copilot Chat. Participants will learn to access Copilot Chat, write effective prompts, detect and prevent AI hallucinations, and use Copilot across the Microsoft 365 suite to accomplish real-world military staff tasks.

All exercises use fictitious but realistic scenarios relevant to Indo-Pacific operations. **Do not enter classified or sensitive information into Copilot Chat.**

## Before You Begin

1. Download the [CopilotResources.zip](https://github.com/opsgility/copilotindopacom/blob/main/assets/CopilotResources.zip) file.
2. Save it to your **Desktop**.
3. Right-click the file and select **Extract All…** to unzip it to your Desktop.

These files are used throughout the exercises — have them ready before starting Exercise 1.

## Prerequisites

- Microsoft 365 work account with Copilot Chat access (GCC High: M365 F1/F3/G3/G5 or Office 365 F1/F3/G3/G5)
- Web browser (Microsoft Edge recommended)
- Access to Microsoft Teams, Word, Excel, Outlook, and PowerPoint
- Web grounding enabled by admin ("Allow web search in Copilot" policy)
